March 10, 2025 | By: Sören Schmidt

Today 3.1.5 has been released for SLES for SAP Applications 12 SP5 and 15 SP3-6. It contains some small fixes:
  • On Azure saptune will now create a drop-in for saptune.service when first called to be started after cloud-final.service. This is needed, because on Azure swap space will be created via cloud-init and some parameters tuned by saptune depend on the amount of swap space.
  • Fix a bug in the NVMe device detection.
  • Fix behavior if limits.conf is missing.
  • Fix value for grub transparent_hugepage entries in SAP Note 2684254.
That's all.
