January 07, 2025 | By: Sören Schmidt

Happy New Year!

A fresh new year and a fresh saptune update as well. Today version 3.1.4 showed up in the repositories of SLES for SAP Applications 12 SP5 and 15 SP2-6, which contains only updated SAP Notes but no fixes.

We have updated each SAP Note to the latest version date and finally replaced the old launchpad links with https://me.sap.com/ But even if we updated the Notes to the latest version, this does not necessarily means a change of tuning. Notes gets updated for various reasons like fixing typos or rephrasing texts. We updated the version even nothing has changed in the tuning, so you can see which latest version we have currently implemented.

In regards of tuning, the following has changed:

  • 1680803 - Sybase - SAP Adaptive Server Enterprise
    With the lates documentation referenced the SAP Note, the only tuning left is setting net.ipv4.tcp_keepalive_time.
  • 2382421 - Optimizing the Network Configuration on HANA- and OS-Level
    Setting net.core.somaxconn is only relevant on SLES 15 releases before SP4.
  • 2578899 - SUSE Linux Enterprise Server 15: Installation Note
    We added the requirement to the presence of the packages polkit and insserv-compat as well as extended starting the sysctl-logger.service to the upcoming SLES 15 SP7
  • 2684254 - HANA DB settings
    The recommendation for trasparent huge pages has changed for SLES 15 SP5 and later. Also the CPU tuning has been limited to x86_64.
  • 3024346 - Linux Kernel Settings for NetApp NFS
    Setting sunrpc.tcp_max_slot_table_entries has been added and a reminder about NFSv3 has been removed.

And by the way. The saptune chapter in the SLES for SAP Application Guide has been reworked, to be more complete and detailed. Maybe you find time to drop by and have a look.

Have a nice day!
